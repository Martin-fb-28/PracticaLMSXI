# PracticaLMSXI

Portfolio Web Personal
Este proyecto es una web de presentación personal y profesional diseñada para mostrar mi trayectoria académica, habilidades técnicas y los proyectos realizados durante mi formación.

Autor
Martín Freire Bao Estudiante de 1º de DAM (Desarrollo de Aplicaciones Multiplataforma) en Liceo La Paz.

¿En qué consiste la práctica?
El objetivo de este proyecto ha sido la creación de un portfolio digital interactivo que centraliza mi perfil profesional. La web está estructurada en secciones clave:

Perfil y Formación: Resumen de mi enfoque y estudios actuales.

Experiencia: Detalles sobre voluntariados y colaboraciones.

Conocimientos Técnicos: Visualización de habilidades en lenguajes como Java, Python, HTML/CSS y SQL.

Proyectos (Práctica): Galería de trabajos prácticos (como el juego de "Hundir la Flota" y gestión de objetos).

Tecnologías y Fuentes Utilizadas
Para el desarrollo de esta interfaz se han utilizado exclusivamente tecnologías fundamentales de desarrollo web front-end:

HTML5: Para la estructuración semántica del contenido.

CSS3: Para el diseño visual, la disposición de los elementos (layout) y el estilo personalizado de la marca personal.
